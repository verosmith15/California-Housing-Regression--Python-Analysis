# California-Housing-Regression--Python-Analysis
End-to-end Machine Learning project to predict median house values in California using Python. Includes advanced data cleaning, exploratory data analysis (EDA), and feature engineering based on economic indicators

## Project Progress: Phase 1 - EDA & Data Cleaning
- [x] Data Ingestion from cloud sources.
- [x] Exploratory Data Analysis (EDA) using Pandas and Matplotlib.
- [x] Statistical profiling (Mean, Median, Skewness).
- [x] Handling missing values: Successfully imputed 207 missing entries in 'total_bedrooms' using median strategy.
- [x] Initial correlation analysis between Median Income and Home Value.

### Key Finding (Economic Insight)
A strong positive linear correlation was identified between household income and property value. Data also shows a price cap at $500,001, which will be addressed during the modeling phase.
